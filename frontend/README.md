# chatIo

Application mobile de messagerie en temps réel

Projet réalisé à des fins d'apprentissage, pour consolider mes compétences en développement mobile full-stack avec un stack moderne.

---

## Stack technique

**Frontend (mobile)**
- React Native + Expo Router
- TypeScript
- Expo Image, Expo Image Picker

**Backend**
- Node.js + Socket.io (communication temps réel)
- MongoDB + Mongoose
- Argon2 (hashage des mots de passe)
- Cloudinary (stockage des médias)

---

## Fonctionnalités

- Authentification (inscription / connexion)
- Conversations directes (1 pour 1)
- Conversations de groupe
- Envoi de messages en temps réel
- Envoi d'images (upload Cloudinary)
- Affichage de l'avatar et du nom des participants
- Tri des conversations par dernier message
- Réactions messages

---

## Objectifs pédagogiques

- Maîtriser la communication temps réel avec **Socket.io** (events, rooms, listeners)
- Approfondir **TypeScript** dans un contexte full-stack (typage des payloads socket, génériques)
- Travailler la gestion d'état React Native dans un contexte asynchrone (socket + état local)
- Manipuler **Mongoose** (populate, timestamps, relations entre collections)
- Gérer l'upload de fichiers vers un service cloud (Cloudinary)
- Structurer proprement une app mobile avec **Expo Router** (layouts, modals, navigation par paramètres)
